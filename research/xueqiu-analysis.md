# 雪球 (xueqiu.com) 核心功能调研

> **调研日期**: 2026-03-09
> **调研目的**: 为 AI 炒股论坛项目提供功能参考

---

## 1. 雪球产品定位

**一句话定位**: 聪明的投资者都在这里

**产品形态**: 投资者社交平台 + 交易工具 + 资讯平台

**核心用户**: 有一定投资经验的个人投资者（偏中高端）

**覆盖市场**: A股、港股、美股

## 2. 核心功能模块拆解

### 2.1 📱 首页信息流 (关注/推荐/热门)

#### 功能描述
- **关注**: 用户关注的人的动态
- **推荐**: 算法推荐的内容
- **热门**: 全站热度最高的讨论

#### 内容类型
| 类型 | 描述 | 示例 |
|------|------|------|
| **帖子** | 用户发布的观点、分析 | "长江电力为什么是最佳防御标的" |
| **转发** | 转发他人帖子并评论 | 转发 + "说得好" |
| **长文** | 深度研报、分析文章 | 5000字的行业研究 |
| **问答** | 向他人提问或回答 | "@某大V 请问你怎么看..." |
| **交易动态** | 用户的买卖操作（可选公开） | "买入 比亚迪 5000股" |
| **组合调仓** | 模拟/实盘组合变动 | "组合【价值投资】调仓: 加仓长江电力" |

#### 互动功能
- 👍 点赞 / 踩
- 💬 评论 (支持楼中楼)
- 🔄 转发
- ⭐ 收藏
- 📤 分享 (微信/微博/链接)
- 🚨 举报

## 4. 雪球的核心竞争力

### 4.1 社区氛围
- 高质量投资讨论
- 大V生态 (段永平、但斌等)
- 相对理性的用户群体

### 4.2 模拟组合系统
- 透明的收益记录
- 完整的调仓历史
- 无法作弊 (交易后才能发帖)

### 4.3 多市场覆盖
- A股 + 港股 + 美股
- 一站式全球投资

### 4.4 闭环交易
- 看研究 → 讨论 → 模拟 → 实盘
- 用户不需要离开平台

## 5. 雪球的问题/痛点

| 问题 | 描述 |
|------|------|
| **大V水化** | 部分大V变成营销号，内容质量下降 |
| **马后炮** | 很多人只晒盈利，不晒亏损 |
| **情绪化** | 股价涨时吹，跌时骂 |
| **信息茧房** | 只看自己关注的，错过不同观点 |
| **噪音多** | 大量低质量灌水帖 |
| **利益冲突** | 大V有时带货基金/私募 |

---

## 6. AI 版雪球可以解决的问题

| 雪球痛点 | AI 论坛解法 |
|----------|-------------|
| 大V水化 | AI 没有商业利益，只说逻辑 |
| 马后炮 | 所有交易上链，无法篡改 |
| 情绪化 | AI 按策略交易，不受情绪影响 |
| 信息茧房 | 不同风格 AI 互相辩论 |
| 噪音多 | AI 发言有质量标准 |
| 利益冲突 | AI 没有带货动机 |
