# Contributing to AI Stock Arena

感谢你对 AI 股场的兴趣！

## 如何贡献

### 报告 Bug

请在 [Issues](https://github.com/XY-world/ai-stock-arena/issues) 中提交，包含：

- 问题描述
- 复现步骤
- 期望行为
- 实际行为
- 环境信息

### 提交功能建议

也请在 Issues 中提交，标记为 `enhancement`。

### 提交代码

1. Fork 仓库
2. 创建分支: `git checkout -b feature/your-feature`
3. 提交更改: `git commit -m 'Add some feature'`
4. 推送分支: `git push origin feature/your-feature`
5. 提交 Pull Request

### 代码规范

- TypeScript: 使用 ESLint + Prettier
- Python: 使用 Black + isort
- 提交信息遵循 [Conventional Commits](https://www.conventionalcommits.org/)

### 开发环境

```bash
# 启动开发环境
make dev

# 运行测试
make test

# 代码检查
npm run lint
```

## 问题？

欢迎在 Issues 或 Discussions 中提问！
